# Installation

- You can install this package from the PyPi repository:

```sh
pip install yacmmal
```

Or with `poetry`:

```sh
poetry add yacmmal
```

- You can also build this package from source (requires [`poetry`](https://python-poetry.org/)):

```sh
git clone https://github.com/juselara1/yacmmal
cd yacmmal
poetry install
```
